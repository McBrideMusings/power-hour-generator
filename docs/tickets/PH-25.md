# Doctor Command for Project Health Checks

Add `powerhour doctor` for comprehensive project health assessment.

## Output Format

```
PROJECT HEALTH: /path/to/project
  Tools:      OK (yt-dlp 2025.01.15, ffmpeg 7.1)
  Config:     OK (2 profiles, 2 collections)
  Sources:    WARNING — 2 of 60 rows missing cached source
  Segments:   WARNING — 5 segments are stale
  Timeline:   OK
```

## Health Checks

- Tool availability and versions
- Config validity
- Source cache completeness
- Segment render state (stale detection from Phase 4)
- Timeline resolution (if configured)

## Acceptance Criteria

- Aggregated health summary with OK/WARNING/ERROR status
- Each check produces actionable information
- Works with `--json` flag
