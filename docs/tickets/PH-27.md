# Export Command for Structured JSON Output

Add `powerhour export` for exporting project data as structured JSON.

## Command Interface

```bash
powerhour export --project <dir> --format json > plan.json
powerhour export --project <dir> --format json --timeline > timeline.json
```

## Acceptance Criteria

- Produces structured JSON with all plan data
- `--timeline` includes resolved timeline
- Output suitable for external tool consumption
