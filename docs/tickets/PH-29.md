# REST API Layer Wrapping Go Functions

Build the REST API that wraps existing Go functions for the web UI.

## API Endpoints

```
GET    /api/project                     # Status + config
GET    /api/project/plan                # Parsed plan rows
PUT    /api/project/plan                # Update plan (write CSV)
GET    /api/project/timeline            # Resolved timeline
POST   /api/project/fetch               # Start fetch (job ID)
POST   /api/project/render              # Start render (job ID)
POST   /api/project/concat              # Start concat (job ID)
WS     /api/jobs/:id/stream             # Real-time progress
GET    /api/segments/:name/preview      # Stream segment video
GET    /api/library/sources             # List sources (paginated)
POST   /api/library/verify              # Start integrity check
POST   /api/library/prune               # Start prune
```

## Acceptance Criteria

- All operations use the same Go functions as CLI
- WebSocket for real-time progress updates
- Proper error handling and status codes
