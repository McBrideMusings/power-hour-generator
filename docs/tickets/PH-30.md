# Frontend - Project Editor

Build the web frontend for project management.

## Pages

- **Overview**: project status, quick actions (fetch all, render all, concat)
- **Plan editor**: table view with inline editing, drag-and-drop reordering, status indicators
- **Timeline view**: visual playback order with duration bars
- **Segment preview**: in-browser video preview of rendered segments
- **Config editor**: form-based editing with live overlay preview
- **Progress monitoring**: real-time fetch/render/concat progress

## Acceptance Criteria

- Plan viewable and editable in browser
- Timeline visualized with correct ordering
- Fetch/render/concat triggerable from UI with real-time progress
- Segments previewable in-browser
