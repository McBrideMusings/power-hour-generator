# Web UI Server Infrastructure

Build the embedded web server for `powerhour serve`.

## Architecture

```bash
powerhour serve --project <dir> [--port 8080]
```

- Embedded web server using Go's `net/http`
- Static frontend assets embedded via `embed` package (single binary)
- Localhost only by default

## Acceptance Criteria

- `powerhour serve` starts a local web server
