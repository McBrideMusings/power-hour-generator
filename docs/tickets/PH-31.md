# Frontend - Media Library Browser

Build the web frontend for browsing the shared media library.

## Pages

- **Library overview**: total sources, disk usage, format breakdown
- **Source browser**: searchable list with thumbnails and metadata
- **Source detail**: full metadata, video preview, project references
- **Library management**: verify integrity, prune unreferenced, manual download

## Acceptance Criteria

- Library browsable with search and thumbnails
- Source detail shows full metadata and preview
- Management actions triggerable from UI
